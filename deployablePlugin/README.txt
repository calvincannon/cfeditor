PREREQUISITES:
* Install a JDK or JRE -- the minimum required version is 5.0.
 * http://wiki.eclipse.org/PDT/Installation#Install_a_JVM

* Download and unpack the Platform Runtime Binary
 * http://download.eclipse.org/eclipse/downloads/drops/R-3.5.2-201002111343/index.php#PlatformRuntime

USAGE:
To use this plugin just put the *.jar file in the plugin directory of your eclipse installation and restart eclipse.
